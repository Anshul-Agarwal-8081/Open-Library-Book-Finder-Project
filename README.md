# Open-Library-Book-Finder-

A sleek, responsive web application designed to help book lovers discover their next great read. This project leverages the Open Library API to provide a searchable database of millions of books, featuring real-time filtering, sorting, and a custom "Reading Goal" tracker.

# Purpose
The goal of this application is to provide a seamless user experience for exploring literary works. It serves as a practical demonstration of asynchronous JavaScript (Fetch API), complex data manipulation using Higher-Order Functions (HOFs), and modern UI/UX design principles.

# Tech Stack
   a-> HTML5: Semantic structure for accessibility.
    
   b-> CSS3 / Tailwind CSS: Custom styling with a focus on a "Bookshelf" grid layout.
    
   c-> JavaScript (ES6+): Core logic, API integration, and DOM manipulation.
    
   d-> Open Library API: The primary data source for book metadata and covers.

# Key Features
   a-> Live Search: Find books instantly by title, author, or ISBN.

   b-> Dynamic Sorting: Organize search results by "Publish Year" (Newest/Oldest) or alphabetically.

   c-> Advanced Filtering: Narrow down results based on specific criteria using JavaScript filter().

   d-> Interactive "Bookshelf": A grid-view UI where users can see book covers and flip them to read descriptions.

   e-> Reading Goal Counter: A persistent counter that tracks books marked as "Read" during the session.

   f-> Debounced Input: Optimized API calling to ensure smooth performance during typing.
